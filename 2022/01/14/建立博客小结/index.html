#博客创建过程:
实现过程比较简单,有手就行.也就尝试了n遍😅

个人博客搭建
1.下载nodejs
2.注册github账号,创建名称为:username.github.io的仓库
3.下载git for windows
4.gitssh配置:双击gitbash
输入命令:ssh-keygen -t rsa -C “username@qq.com“
打开ssh设置添加

5.hexo使用
npm install hexo-cli -g
hexo init hjmblog
cd hjmblog
npm install hexo s预览

6.发布到博客
1.配置config中的deploy
2.type : git
3.repo: ……
4.url : ……
5.hexo-deployer-git –save
6.修改branch: master
7.git config –global user.email “username@qq.com“
git config –global user.name “username”
8.hexo g
hexo d

到这里你就成功了,怎么样是不是很简单,现在就去试试吧.
此处省略万字吐槽😅

补充几个更新github仓库的指令：

git status 查看状态

git add . 添加文件

git commit -m “xxx” 编辑提交信息

git push 提交文件
